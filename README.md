# GeneseBot

My first chatbot with some functionalities

## Install

<pre>
  git clone https://github.com/gabrielrufino/GeneseBot<br>
  cd GeneseBot<br>
  npm install<br>
</pre>

## Commands list

/start - Call the bot<br>
/help - Help message<br>
/date - Current date and time<br>
/bitcoin - Current info about bitcoin
